# AnalyticalCore

AnalyticalCore is the core analytical processing unit of the Market Prediction System (MPS).

## Structure
- `DataIngest`: Handles importing, fetching, and feeding of raw market data.
- `DataProcessing`: Manages cleaning, transformation, and normalization of ingest data.
- `DataGating`: Controls access, filtering, validation, and gating logic for downstream consumers.

## Requirements
- Python 3.12+
